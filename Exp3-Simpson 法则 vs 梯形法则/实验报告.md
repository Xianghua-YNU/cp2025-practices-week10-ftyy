1. 理解并掌握 Simpson 法则与梯形法则的数值积分原理及实现方法。
2. 通过实验比较两种方法在积分精度和误差方面的表现。
3. 能够分析不同子区间数 $N$ 对积分结果的影响。

## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
其基本思想是用二次多项式（抛物线）来近似被积函数在小区间上的曲线，从而利用抛物线的积分来近似原函数的积分。相比于梯形法则（用直线近似），Simpson 法则在光滑函数上通常能提供更高的精度。N 必须为偶数：因为 Simpson 法则每次需要两个子区间（三个点）来构造抛物线，所以总子区间数N 必须是偶数。函数足够光滑：Simpson 法则要求被积函数f(x) 在积分区间上足够光滑（至少四阶可导），因为其误差项依赖于f(x)。如果函数不光滑（如存在尖锐变化或间断点），Simpson 法则的精度可能会下降。积分区间有限：Simpson 法则适用于有限区间 [a,b] 上的积分。对于无穷积分或其他特殊情况，需要结合其他方法（如截断或变量替换）。
![image](https://github.com/user-attachments/assets/890c0a50-2c97-4a10-bf76-bc6daad0cfef)

$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
用直线代替曲线：将被积函数f(x) 在积分区间 [a,b] 上的曲线用一系列直线段（即梯形）近似。分段线性逼近：将积分区间划分为若干个子区间，在每个子区间上用线性插值（两点直线）代替原函数，计算梯形的面积之和作为积分的近似值。
![image](https://github.com/user-attachments/assets/d854d5c4-2be6-4ab5-bde6-48150cf8c4a1)

$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

## 三、实验代码

请粘贴你的完整 Python 实现代码（包括 Simpson 法则和梯形法则）：

```import numpy as np

# 待积分函数
def f(x):
    return x**4 - 2*x + 1

# 已给出的梯形法则积分函数，供参考比较用
def trapezoidal(f, a, b, N):
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    fx = f(x)
    integral = h * (0.5 * fx[0] + np.sum(fx[1:-1]) + 0.5 * fx[-1])
    return integral

# Simpson 法则积分函数
def simpson(f, a, b, N):
    if N % 2 != 0:
        raise ValueError("Simpson 法则要求 N 必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    fx = f(x)
    # 奇数索引（1,3,5,...,N-1）
    odd_sum = np.sum(fx[1:N:2])
    # 偶数索引（2,4,6,...,N-2）
    even_sum = np.sum(fx[2:N:2])
    integral = h / 3 * (fx[0] + 4 * odd_sum + 2 * even_sum + fx[N])
    return integral

def main():
    a, b = 0, 2
    exact_integral = 4.4

    for N in [100, 1000]:
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)

        trapezoidal_error = abs(trapezoidal_result - exact_integral) / exact_integral
        simpson_error = abs(simpson_result - exact_integral) / exact_integral

        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

if __name__ == '__main__':
    main()
```

## 五、结论
- 总结两种方法的适用场景和优劣。
- 梯形法则（Trapezoidal Rule）
适用场景：
函数不够光滑：当 f(x) 存在尖锐变化、间断点或高阶导数较大时，梯形法则更稳健。
快速估算：需要低计算成本的初步近似（如实时计算或嵌入式系统）。
非周期性函数：适用于无显著周期性振荡的函数（如分段线性函数）。
优势：
计算简单：只需计算函数值并加权求和，适合大规模数据或并行计算。
稳定性强：对非光滑函数的适应性优于高阶方法（如 Simpson 法则）。
无需偶数区间：可自由选择分割数 
劣势：
精度较低：误差为二阶小量
不适合高振荡函数：对快速变化的函数（如高频信号）近似效果差。
Simpson 法则（Simpson's Rule）
适用场景：
光滑函数：当 f(x) 至少四阶可导且高阶导数较小时（如多项式、指数函数、三角函数等）。
高精度需求：需要快速收敛的解（如科学计算或工程仿真）。
周期性函数：对振荡函数（如 sin(x)、cos(x)）的积分效果优异。
优势：
精度高：误差为四阶小量，相同N 下精度显著高于梯形法则。
效率高：较少的计算量即可达到目标精度（如 h 较大时仍能保持高精度）。
精确积分三次多项式：即使使用二次逼近，仍能精确计算三次多项式的积分。
劣势：
要求偶数区间：
N 必须为偶数，限制了分割的灵活性。
对非光滑函数敏感：若 f(x) 存在间断点或尖锐变化，误差可能急剧增大。
计算稍复杂：需区分奇偶节点的权重（4 和 2 的交替系数）。
- 结合本实验结果，说明实际工程中如何选择数值积分方法。
当对误差精度要求较高时用Simpson法，反之用梯形积分法。
