## 二、实验原理

### 2.1 梯形积分法

对于离散的速度数据点$(t_0,v_0),(t_1,v_1),...,(t_N,v_N)$，总距离（位移）可通过梯形法则计算：

$$
\text{距离} \approx \sum_{i=0}^{N-1} \frac{v_i + v_{i+1}}{2} (t_{i+1} - t_i)
$$

### 2.2 累积距离计算

使用`scipy.integrate.cumulative_trapezoid`计算每个时间点的累积距离：

$$
\text{累积距离}(t_i) = \int_{t_0}^{t_i} v(t)dt
$$

```python

import numpy as np
from scipy.integrate import cumulative_trapezoid
import matplotlib.pyplot as plt
import os

def main():
    try:
        # 1. 获取数据文件路径（使用相对路径）
        data_file = os.path.join(os.path.dirname(__file__), 'Velocities.txt')  # 假设数据文件名为data.txt

        # 2. 读取数据（使用numpy.loadtxt）
        data = np.loadtxt(data_file, delimiter='\t')  # 假设数据以逗号分隔
        t = data[:, 0]  # 时间列
        v = data[:, 1]  # 速度列

        # 3. 计算总距离（使用numpy.trapz）
        total_distance = np.trapz(v, t)
        print(f"总运行距离: {total_distance:.2f} 米")

        # 4. 计算累积距离（使用cumulative_trapezoid）
        distance = cumulative_trapezoid(v, t, initial=0)

        # 5. 绘制图表
        plt.figure(figsize=(10, 6))
        plt.plot(t, v, 'b-', label='Velocity (m/s)')
        plt.plot(t, distance, 'r--', label='Distance (m)')
        plt.title('Velocity and Distance vs Time')
        plt.xlabel('Time (s)')
        plt.ylabel('Velocity (m/s) / Distance (m)')
        plt.legend()
        plt.grid(True)
        plt.show()

    except FileNotFoundError:
        print("错误：找不到数据文件")
        print("请确保数据文件存在于正确路径")

if __name__ == '__main__':
    main()
```

### 4.2 误差分析
分析梯形积分法计算距离的误差来源
**近似替代误差**
梯形积分法的基本思想是将积分区间分成若干个小区间，然后在每个小区间上用梯形的面积来近似代替函数与坐标轴所围成的面积。
然而，函数的图像通常不是直线，而是曲线，用梯形去近似曲线下方的面积，必然会产生误差。这种误差是由于用线性函数（梯形的边）来近似非线性函数（被积函数）所导致的，被积函数的非线性程度越高，近似替代产生的误差就越大。
**区间划分误差**
梯形积分法的精度与积分区间的划分有关。一般来说，划分的区间越多，近似程度就越高，误差就越小。
如果区间划分得不够细，那么在每个小区间上用梯形近似代替函数曲线下方面积的误差就会较大。相反，若要提高精度，就需要增加区间数量，但这会增加计算量。
**端点值误差**
在梯形积分法中，每个小区间的端点值被用来计算梯形的面积。如果被积函数在端点处的值不准确，或者端点处的函数值不能很好地代表该区间内函数的平均行为，那么就会引入误差。
例如，当函数在端点处有间断点、尖点或者剧烈变化时，仅用端点值来计算梯形面积就不能准确反映函数在该区间上的积分值。
